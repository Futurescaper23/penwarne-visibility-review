# 3DVista Hotspot URLs

Use these preferred bridge links for 3DVista hotspots so the images open inside the main app modal over the embedded tour instead of as raw files.

These links do not depend on 3DVista correctly targeting the parent window. The bridge page updates the parent app when the tour is embedded, and falls back to the standalone polished viewer if it is opened on its own.

- Preferred pattern  
  `/viewer-bridge.html?src=/projects/...&title=...&caption=...`
